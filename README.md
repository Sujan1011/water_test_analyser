 # Water Quality Test Analyzer

A comprehensive Python application for analyzing paper-based water testing strips using computer vision.

## Features

- 📸 Capture images from camera or upload from gallery
- 🧪 Support for pH, Iron, and Nitrate testing
- 🔬 Scientific explanations for color changes
- ⚠️ Safety recommendations based on results
- 📊 Color intensity analysis with statistics
- 💾 Export results to JSON or text reports
- 🌐 Web interface option for remote access
- 🤖 Machine learning color classification
